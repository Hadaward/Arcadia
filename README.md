# Arcadia
Arcadia is a voice-powered magic framework for Hytale.

Players can cast spells through spoken incantations using a configurable grammar system composed of Elements, Forms, Modifiers and Targets.

## Features
- Voice recognition using Vosk
- Configurable spell grammar
- Extensible spell definitions
- Custom spell creation
